=========================================
    HOW TO ADD NEW PORTFOLIO IMAGES:

    1. Put your .jpg file in: public/images/portfolio/
    2. Add a new line under the right category:
       Image: filename.jpg | Brand Name | Category • Description
    3. Save this file and run the commands below

    HOW TO ADD TESTIMONIAL PROFILE PICS:

    1. Put the photo in: public/images/profiles/
    2. Name it: client-1.jpg, client-2.jpg, etc.
    3. Add a Photo line under the testimonial:
       Photo: client-1.jpg
